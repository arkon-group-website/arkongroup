# ARKON Group Codex Build Pack

Website: https://arkongroup.com.sa  
Build stack: WordPress + Bricks Builder + custom CSS/JS + optional Spline/Three.js 3D sections  
Brand direction: Premium dark blue + silver + gold, 3D Saudi project command-center style.

## Included

- `AGENTS.md` — main Codex instructions.
- `docs/website-blueprint.md` — sitemap, homepage plan, SEO, structure.
- `docs/homepage-copy-en-ar.md` — homepage copy in English and Arabic.
- `docs/3d-hero-concept.md` — creative and technical direction for the hero scene.
- `arkon-child-theme/` — starter WordPress child theme files.
- `bricks-sections/` — Bricks-ready HTML section snippets.
- `content/` — structured content JSON.
- `schema/` — starter JSON-LD.
- `codex-prompts/` — task prompts for Codex.
- `assets/3d-briefs/` — Spline/3D creative brief.

## How to use

1. Create a private GitHub repository called `arkon-group-website`.
2. Upload this pack to the repository.
3. Install WordPress and Bricks Builder on a staging domain.
4. Install the child theme under `wp-content/themes/arkon-child-theme`.
5. In Bricks, create the homepage and import/adapt snippets from `bricks-sections`.
6. Ask Codex to follow `AGENTS.md` and execute tasks from `codex-prompts/`.
7. Review every visual output manually before publishing.
