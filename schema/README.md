# ARKON Group Schema Patterns

These files are reusable JSON-LD patterns for WordPress, Bricks Builder, or an SEO plugin such as Rank Math.

- `organization.json` is the conservative homepage Organization entity for ARKON Group and the five confirmed companies.
- `website.json` connects the website entity to the Organization entity and supports English and Arabic.
- `breadcrumb-list.json` is a company-page breadcrumb example. Adapt the last item for the current page.
- `service.json` is a service-page example using engineering consultancy. Adapt names, URLs, descriptions, and related company references to the visible page copy.
- `faq-page.json` should only be used on pages where the same FAQ questions and answers are visible to users.
- `professional-service.json` is a company-page pattern for service businesses where LocalBusiness / ProfessionalService markup is appropriate.

Do not add approvals, certifications, awards, rankings, client logos, authority status, exact address, phone number, opening hours, ratings, or social profiles unless those claims are supported by project files.
