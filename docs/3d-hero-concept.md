# 3D Hero Concept — ARKON Project Command Center

## Concept

A premium dark 3D command-center interface representing ARKON Group as an integrated Saudi project support ecosystem.

## Scene

- Dark navy background.
- Silver grid floor.
- Subtle Saudi geometric pattern.
- ARKON Group logo floating in the center.
- Five company modules around it.
- Blue/gold data lines connecting all modules.
- Abstract Saudi skyline / mega-project blocks in the distance.
- Soft glow and glassmorphism panels.

## Modules

1. AEC — blueprint / engineering grid / structural module
2. ELITE — shield / secure command module
3. IBS — building / construction module
4. QIC — testing lab cube / soil layer module
5. TP Digital — digital signal / campaign dashboard module

## Animation sequence

0-1s: grid fades in  
1-2s: ARKON Group mark appears  
2-4s: five modules animate into place  
4-6s: connecting lines activate  
Scroll: scene slowly shifts downward into group company cards

## Technical implementation

Use Spline for v1 hero. Export a static fallback PNG/WebP for mobile. Use lazy loading and reduced-motion handling.

## Performance rule

Do not load 3D scene on low-power mobile devices. Use static fallback image and CSS-only glow effect.
